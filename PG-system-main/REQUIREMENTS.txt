# PG Management System - Requirements & Dependencies

## System Requirements

- **OS:** Windows 10+, macOS 10.15+, or Linux (Ubuntu 18.04+)
- **Node.js:** 18.0.0 or higher (20.x LTS recommended)
- **Package Manager:** pnpm 8+ (npm/yarn also supported)
- **Git:** 2.30.0 or higher
- **RAM:** 1GB minimum (2GB for building)
- **Disk:** 500MB for app + dependencies

## Browser Support

- Chrome 90+
- Firefox 88+
- Safari 14+
- Edge 90+
- Mobile: iOS 14+, Android 10+

## Installation

### Option 1: pnpm (Recommended)
```bash
pnpm install
```

### Option 2: npm
```bash
npm install
```

### Option 3: yarn
```bash
yarn install
```

## Environment Setup

Create `.env.local` with:

```
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-public-key
SUPABASE_SERVICE_ROLE_KEY=your-secret-key
NEXT_PUBLIC_VERCEL_ANALYTICS_ID=optional
```

## Build Requirements

- Build time: 2-3 minutes
- Build output: ~200-400MB (.next folder)
- Requires 2GB RAM during build

## Runtime Requirements

- Memory: 512MB minimum, 1GB recommended
- Disk: 500MB for application files
- Bandwidth: Varies by usage

## Performance Targets

- First Contentful Paint (FCP): < 2.5s
- Largest Contentful Paint (LCP): < 4s
- API Response Time: < 200ms average
- Database Queries: < 100ms average

## Troubleshooting

### Clear everything and reinstall
```bash
rm -rf node_modules pnpm-lock.yaml .next
pnpm install
pnpm build
```

### Check security vulnerabilities
```bash
pnpm audit
pnpm audit --fix
```

### Update all dependencies
```bash
pnpm upgrade --latest
```

## Security Notes

- Keep `.env.local` in `.gitignore` (NEVER commit)
- Service Role Key must ONLY be used on backend
- Enable HTTPS for production
- Implement rate limiting on APIs
- Validate all user inputs
- Keep dependencies updated

## Deployment Checklist

- [ ] All dependencies installed: `pnpm install`
- [ ] Environment variables set in `.env.local`
- [ ] Database configured and migrated
- [ ] Build successful: `pnpm build`
- [ ] No lint errors: `pnpm lint`
- [ ] Tested locally: `pnpm start`
